发表于 2018-07-06 | 分类于 沉淀

终于回来了，但是不知道自己内心是否已归。

无论怎样，我要up,up,up.

趁热打铁，来过一遍现在自己理解的Kotlin里面高阶函数的使用场景吧。
现在对与kotlin还停留在表层，可能是因为自己对Java也深入了解得不够的原因吧。所以，根据目前自己了解到的五个高阶函数做做记忆的加固。

  • let:允许，任由，让，随；假设；出租
  • also:也；同样；并且；另外
  • apply:申请；涂；应用，运用；敷；请求，适用，适合
  • run/T.run:使奔跑，运行，经营
  • with:随着，和，跟
